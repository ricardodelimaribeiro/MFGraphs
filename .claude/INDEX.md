# Jamaratv9 Pruning Statistics Project — Complete Index

## Overview
This directory contains all preparation, documentation, and instrumentation for extracting and measuring Jamaratv9 solver pruning statistics (2^95 alternatives, branch elimination breakdown, timing analysis).

All **Phases 0-5 completed**. Ready for user to run measurements on their machine.

### `Scripts/` — Executable Code

#### Main Instrumentation (Ready to Run)
- **`InstrumentJamarat.wls`** ✅ PRODUCTION READY
  - Non-invasive wrappers for: CriticalCongestionSolver, TripleClean, DNFReduce, ReduceDisjuncts
  - Accumulates: q, branch counts, timings, pruning stats
  - Exports JSON to `Results/jamaratv9_pruning_stats.json`
  - Run time: ~12 minutes
  - Command: `wolframscript -file Scripts/InstrumentJamarat.wls`

#### Template Scripts (User Creates From .md)
- **`ValidateInstrumentation.wls`** (template in PHASE3_VALIDATION.md)
  - User creates this from code block in Phase 3
  - Tests on fork1 & merge1 examples
  - Verifies < 10% overhead before Phase 4
  - Run time: ~2-5 seconds each

- **`GenerateStatsTables.wls`** (template in PHASE4_EXECUTION.md)
  - User creates this from code block in Phase 4
  - Reads `Results/jamaratv9_pruning_stats.json`
  - Generates LaTeX statistics table
  - Prints summary metrics
  - Run time: ~1 second

#### Reference (Optional)
- **`Phase0_Extract.wls`** (for reference only)
  - Automated Phase 0 data extraction
  - Was used to verify assumptions

### `Results/` — Output Directory (User Creates)

Created **after Phase 4 solver run**:

- **`jamaratv9_pruning_stats.json`** 📊 MACHINE-READABLE
  - q (uncertain triples): measured value
  - branchesEntered, branchesPrunedImmediate, branchesPrunedElimination, leavesSurviving
  - totalWallTime
  - passTimings array (TripleClean breakdown)
  - reduceDisjunctsStats (pre/post reduction counts)

- **`jamaratv9_run.log`** (optional, from stdout redirect)
  - Detailed solver progress & timing messages

- **`jamaratv9_stats_table.tex`** (generated by GenerateStatsTables.wls)
  - LaTeX pruning statistics table
  - Ready to copy into main.tex §5.3
